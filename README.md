# UEminiMapPlugin

Minimap System Plugin 为 Unreal Engine 项目提供了一个完全可自定义、可扩展的小地图与按键地图系统框架。
该插件支持动态图标管理、战争迷雾渲染，以及在运行时与编辑器中都可使用的精确世界坐标到地图坐标同步机制。

🔹 核心功能
![Map Preview](./gif/4putIcon.gif)

图标组件系统
轻量级的 UMapIconComponent 可挂载到任意 Actor 上。
每个图标都会自动注册并在 小地图 与 按键地图 上同步更新位置。
![Map Preview](./gif/2keymap.gif)
世界坐标与地图坐标转换
小地图固定玩家（或相机）于中心位置，
通过材质计算世界空间偏移实现位置映射。
按键地图使用相同的坐标逻辑，以保证视觉一致性。
![Map Preview](./gif/3minimap.gif)
战争迷雾支持
小地图与按键地图均支持实时的战争迷雾渲染。
已探索区域通过基于 Render Target 的材质系统实时显示。

实时图标跟随
所有图标都会平滑地跟随其世界中对应的对象移动，
并根据玩家位置自动更新可见性与位置。
![Map Preview](./gif/1changeMap.gif)
室内房间切换地图显示
由大地图进入到房间后，切换室内地图显示

🔹 适用场景

非常适合：

RPG、策略或探索类游戏

需要同时显示小地图与大地图的项目

含有探索或可见性解锁机制的系统

🔹 技术特点

基于 UMG、材质 Render Target、支持 Blueprint/C++ 扩展

运行时与编辑器下均可实时预览

结构清晰、模块化设计，方便集成进现有项目
